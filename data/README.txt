Permanent meeting materials live here as plain .txt files (lightweight, loaded on demand).

Primary files (edit these for interview content):
  about-me.txt       — About Me / resume
  work-history.txt   — Work Experience
  interview-qa.txt   — Interview Q&A
  personal-life.txt  — Personal Life Context

Legacy filenames (about me.txt, etc.) are kept for compatibility; run npm run embed-base to sync.

Do NOT place large audio, PDF, or Excel files in this folder — they slow down the browser extension.
Move bulky assets outside the extension folder (e.g. ../meeting-assist-user-files/).
